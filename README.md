# LottoNum_Game

# 소개 
----------------------------
C언어로 구현한 간단한 로또번호 맞추기 프로그램 입니다. 

사용자가 총 6개의 번호를 입력 후 자동으로 생성된 7개의(보너스 번호 포함) 번호를 맞춘 갯수에 따라 등수가 나눠지는 프로그램 입니다.
